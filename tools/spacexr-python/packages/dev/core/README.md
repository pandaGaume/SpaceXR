# SpaceXR Python Port

This project is a Python port of the TypeScript library "SpaceXR", originally designed for various spatial calculations and experiments within the realm of space exploration simulations. The Python version aims to explore different approaches within Python scripting, leveraging Python's rich ecosystem and simplicity for rapid development and experimentation.

## Overview

SpaceXR in Python seeks to offer similar functionality to its TypeScript counterpart, with adaptations and improvements that are idiomatic to Python. This includes handling spatial data, performing calculations related to orbits, spacecraft maneuvers, and other celestial mechanics in a way that's accessible to both enthusiasts and professionals in the field of astronomy and astrophysics.

## Features

-   Spatial Calculations: Perform precise calculations for distances, orbits, and trajectories.
-   Celestial Mechanics: Simulate gravitational forces, orbital dynamics, and other fundamental principles governing bodies in space.
-   Experimentation Platform: Test hypotheses and models related to space exploration and astrodynamics.

## Getting Started

To get started with the SpaceXR Python library, ensure you have Python 3.6 or later installed on your system. You can then install the library using pip:

```bash
pip install spacexr-python
```

### Basic Usage

Here's a quick example to demonstrate how you might use the SpaceXR library:

```python
from spacexr import EPSG3857

# Example: use of tile metrics
metrics = EPSG3857()

```
