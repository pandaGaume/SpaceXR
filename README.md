# spacexr

Support for space and terrain rendering.

## Building

Build the build-tools by running the following command from packages/dev/build-tools
`npm run build:build-tools`

Link @dev/build-tools to the local package (from packages/dev/build-tools again).
`npm link @dev/build-tools`

Build the core package.
`npm run build:core:dev`

Build the babylon package.
`npm run publish:babylon:dev`
